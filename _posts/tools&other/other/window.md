# window



## 快捷键

### 虚拟桌面

| 快捷键                 | 描述         |
| ---------------------- | ------------ |
| `win + ctrl + D `      | 创建虚拟桌面 |
| `win + ctrl + (<-/->)` | 虚拟桌面切换 |
| `win+ ctrl + F4`       | 关闭虚拟桌面 |



### 其他

| 快捷键        | 描述           |
| ------------- | -------------- |
| `win + m`     | 全部窗口最小化 |
| `win + (+/-)` | 放大镜         |
| `win + i`     | 打开设置       |
| `win + e`     | 管理视图       |
| `win + a`     | 操作界面       |
| `win + L`     | 锁定电脑       |
